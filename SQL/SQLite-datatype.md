## 데이터 타입
| Type | Description |
| ------ | ----------- |
| NULL | null 값 |
| INTERGER | 부호있는 정수 (1,2,3,4,6,8 Byte 저장) |
| REAL | 부동 소숫점 숫자 (8 Byte) | 
| TEXT | 텍스트 형식 (UTF-8 or UTF-16 or UTF-16-LE) |
| BLOB | Binary Large OBject, 바이너리 형태 데이터 | 

### int  
Interger 형태 대신에 int로 지정할 수 있다. (Integer 형태로 반환)
